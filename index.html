The Mind Behind the Code

I’m Mohammad Rashim, a software engineer who builds at the intersection of enterprise-grade Java backends, cloud-native delivery, and modern artificial intelligence. I create systems that are fast, reliable, and thoughtfully designed for real users.

My Coding Philosophy: Every line of code should be intentional. I write services that are modular, testable, and self-documenting, bridging the power of Spring Boot microservices with intuitive, responsive frontend interfaces.
